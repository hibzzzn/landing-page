# HB Code - Landing Page

A modern landing page created as part of **The Odin Project - Foundations Course**.

## 📖 About

This project is a landing page for a fictional web development company called **HB Code**.

The goal of this project was to practice:

- HTML5 structure
- CSS styling
- Flexbox layout
- Typography
- Colors
- Spacing (margin & padding)
- Buttons
- Cards
- Sections
- Git & GitHub workflow

## ✨ Features

- Modern navigation bar
- Hero section with call-to-action button
- Services section with four feature cards
- Customer testimonial section
- Call-to-action section
- Footer
- Clean and responsive-looking layout

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- Git
- GitHub
